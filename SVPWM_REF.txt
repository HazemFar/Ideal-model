*generating FOC switching pattern using Vsn voltage injection(triangular wave injection)
.SUBCKT SVPWM_REF A B C AP BP CP



Ran A 0 100MEG
Rbn B 0 100MEG
Rcn C 0 100MEG


.func getmaximum(AN,BN,CN) {max(max(AN,BN),CN)};Get maximum
.func getminimum(AN,BN,CN) {min(min(AN,BN),CN)};Get minimum
Rcalcoffset calc_offset 0 0.5

*Note : -ve sign is the direction of the current source
Goffset  calc_offset 0 value={ getmaximum(V(A),V(B),V(C)) + getminimum(V(A),V(B),V(C))};V_offset=-(Vmax+Vmin)/2




Bvap AP 0 v = V(A)+V(calc_offset)  ;voltage of the pole
Bvbp BP 0 v = V(B)+V(calc_offset)
Bvcp CP 0 v = V(C)+V(calc_offset)


.ENDS SVPWM_REF
